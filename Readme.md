# AviUtl-Shutdown-Plugin
AviUtlの起動から30秒後にPCがシャットダウンするプラグインです。

**作者のやさしさにより**、30秒の猶予があります。  
(``shutdown -a``を用いることにより解除できると思われます。)

シャットダウン用のアニメーション効果スクリプトも同梱しています。  
(こちらも30秒の猶予があり、プラグイン側と同じことをしています。)

疲れすぎた人が突発的な思考により作り出したプラグインです。

**ネタ枠のプラグイン・スクリプトですね。これを導入する意味とは**

GitHub ActionsのWorkflowの方は、他のAviUtlのプラグインのコンパイルにも使えると思いますので是非使ってください。  
(そちらもMITライセンスで頒布いたします。)

どちらかといえば、Workflowがメイン？  
(プラグインを作成した時間より、Workflowに費やした時間のほうが長い)
